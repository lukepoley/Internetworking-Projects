A powerful Android application built with Kotlin that helps you analyze your WiFi network, detect signal dead zones, measure connection speed, and identify potential bandwidth hogs.

## Features

### 1. Signal Detector & Direction Finder
- Real-time **Signal Strength** monitoring (dBm and %)
- **Direction Estimation**: Move around your space, and the app analyzes RSSI trends to tell you if you're getting closer or further from your router.
- Visual signal bars and quality indicators.

### 2. Speed Test
- built-in speed test for **Download** and **Upload** speeds.
- Measures network **Latency** (ping).
- Rates your connection quality.

### 3. Device Discovery
- Scans your local network to find connected devices.
- Identifies device types.
- Highlights likely bandwidth hogs.

## Getting Started

1. Open in Android Studio
2. Build the project
3. Run on a physical Android device (required for WiFi scanning)

## Permissions
- **Location**: Required for WiFi scanning
- **WiFi**: To access network info
- **Internet**: For speed testing
